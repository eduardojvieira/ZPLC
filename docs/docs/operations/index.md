# Operations

Operations for v1.5 focus on release evidence, diagnostics, and controlled recovery.

## Minimum Operator Guidance

- how to run automated release validation
- how to capture human evidence for desktop and HIL gates
- how to recover when deployment or debugging validation fails
- how to rescope unsupported claims instead of silently shipping them

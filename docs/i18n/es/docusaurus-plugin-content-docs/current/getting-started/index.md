# Primeros Pasos

Use esta ruta para una primera validacion honesta de ZPLC v1.5:

1. Instale dependencias con `bun install`.
2. Compile el IDE y la documentacion desde un checkout limpio.
3. Ejecute una simulacion con una muestra canonica.
4. Despliegue a una placa incluida en la lista de soporte de v1.5.

La documentacion canonica amplifica esta ruta y evita afirmaciones fuera del alcance verificado.

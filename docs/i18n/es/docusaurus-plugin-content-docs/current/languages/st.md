# Structured Text (ST)

Structured Text es la base semantica canónica de ZPLC v1.5.

## Papel en v1.5

- `ST` es la referencia de comportamiento.
- `IL`, `LD`, `FBD` y `SFC` son rutas soportadas cuando convergen al mismo contrato.
- La paridad se valida contra la suite canónica de lenguajes.

# Operaciones

La operacion de v1.5 se enfoca en evidencia, diagnostico y recuperacion reproducible.

## Cobertura minima

- validacion automatizada antes del release
- evidencia humana para gates sensibles al entorno
- recuperacion clara ante fallos de despliegue, conectividad o depuracion

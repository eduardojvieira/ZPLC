# Despliegue desde el IDE

El despliegue de v1.5 debe reflejar solo las placas y los metodos validados para el release.

## Flujo esperado

1. Compilar el proyecto
2. Seleccionar una placa de la lista soportada
3. Desplegar por el metodo disponible
4. Confirmar el estado y preparar la sesion de depuracion si aplica

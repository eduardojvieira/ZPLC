# Resumen del IDE

El IDE de ZPLC debe cubrir para v1.5 el flujo completo de autor, compilar, simular,
desplegar y depurar para cada lenguaje reclamado por el release.

## Superficies clave

- edicion de texto para `ST` e `IL`
- editores visuales para `LD`, `FBD` y `SFC`
- simulacion
- despliegue
- depuracion con breakpoints, watch y force values

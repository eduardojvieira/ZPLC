# Desktop Smoke Checklist

Use this checklist on macOS, Linux, and Windows.

- [ ] Install or launch the desktop IDE artifact
- [ ] Open a sample project
- [ ] Compile successfully
- [ ] Run simulation and observe expected state
- [ ] Deploy to a target or simulated runtime
- [ ] Set and hit a breakpoint
- [ ] Step / continue execution
- [ ] Inspect watch values
- [ ] Force a value and observe the result
- [ ] Save evidence artifacts and result notes
